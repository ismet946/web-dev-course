bootstrap grid system

the bootstrap grid system is a responsive layout structure based on on a series of rows and columns.it help to creat a responsive and mobile friendly design without having to write extensive css.
responsive design:bootstraps allows you to adjust the column layout based on screen size examples are below ;
( .col-sm-): for 576px small devices | (.col-md-):for 768px medium devices
(.col-lg-):for 992pc large devices (.col-xl-):for extra large devices
